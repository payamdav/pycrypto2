# Agent Instructions

Every AI agent working in this repository **must read all files** in the `agents/` directory and its subdirectories before performing any task.

The `agents/` directory contains critical rules, instructions, and how-to guides that govern how work must be done in this project. These files are the authoritative source for:

- Mandatory coding and dependency rules
- Package documentation and conventions
- Dataset handling procedures
- Skills — reusable agent roles defined in `agents/skills/`, each describing a specific capability (e.g., `spec_writer`, `code_writer`, `code_review`, `code_tester`)
- Any other project-specific instructions

Failure to read these files before acting may result in incorrect, incomplete, or non-compliant work. Always traverse the full `agents/` hierarchy and internalize every file found there.
